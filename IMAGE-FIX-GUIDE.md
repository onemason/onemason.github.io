## 如何替换占位图片

### 方法1：使用原始图片文件

1. 找到原始图片文件
2. 将图片文件复制到对应的文章资源文件夹
3. 更新文章中的图片引用

例如，对于《痛》这篇文章：
```bash
# 1. 将原始图片复制到资源文件夹
cp /path/to/original/痛.jpg source/_posts/2017/痛/痛/

# 2. 更新文章引用
# 修改 source/_posts/2017/痛.md 中的图片引用：
# 从: ![痛](痛/image-placeholder.svg)
# 到: ![痛](痛/痛.jpg)
```

### 方法2：使用新图片

1. 准备新的图片文件（建议格式：JPG、PNG、WebP）
2. 将图片文件复制到对应的文章资源文件夹
3. 更新文章中的图片引用

### 方法3：保持SVG占位图片

如果您喜欢当前的SVG占位图片，可以保持现状。这些占位图片：
- 明确说明了图片已丢失
- 提供了原始链接信息
- 视觉上清晰明了

## 图片文件命名建议

为了保持一致性，建议使用以下命名规范：

1. **主要图片**: 使用文章标题作为文件名（英文或拼音）
   - `痛.jpg`
   - `toast-this-new-world.jpg`
   - `vue-application.png`

2. **多张图片**: 使用描述性名称
   - `痛-封面.jpg`
   - `vue-流程图.png`
   - `2019-合影.jpg`

3. **文件格式**:
   - 照片: JPG
   - 图表/截图: PNG
   - 矢量图: SVG
